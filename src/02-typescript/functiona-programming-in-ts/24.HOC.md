# High-order Function

https://www.youtube.com/watch?v=1cGPmdcQ2p8

High-order function:

- Takes one or more functions as parameters or
- Returns or function
- Or both of them

```ts
type Filter = <A>(f: A => boolean) => A[] => A[] // not valid, jusr for example

const filter: Filter = f => arr => arr.filter(f);

```

- Lambda function give us concise syntax for small functions.
- Lambda function are useful for defining curried and high-order functions
- Lambda function has limitations
- Lambda function syntax aligns with the function-as-first-citizen perspective
