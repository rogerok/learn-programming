# Group

https://www.youtube.com/watch?v=6atlKUsn0mU

Group - это моноид, в котором у каждого элемента существует единственный обратный элемент
Группа это фундаментальная структура в математике.
Группа кодирует (или выражает) понятие симметрии

```ts
interface Group<A> {
  concat: (x: A, y: A) => A;
  empty: A;
  inverse: (a: A) => A;
}
```

Т.к. группа это моноид, мы можем объявить её так

```ts
interface Group<A> extends Monoid<A> {
  inverse: (a: A) => A;
}
```

Т.е. группа имеет набор действий, который может привести ее к изначальному состоянию. (надо уточнить формулировку)

  empty: 0,
  invserse: (a) => -a,
};

  addGroup.empty,
  addGroup.concat(80, addGroup.concat(20, addGroup.inverse(10))),
