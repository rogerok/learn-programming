# Введение

Браузер заточен под событийную модель - код не исполняется непрерывно, а ожидает пользовательских действий.

- Код не должен блокировать выполнение
- Пользователь должен сохранять возможность взаимодействия с интерфейсом
- Несколько задач могут выполняться практически параллельно

# Почему JavaScript асинхронный

По этой причине все движки JS построены по асинхронной модели.
