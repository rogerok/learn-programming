# 2.1.6. Введение в булеву алгебру

Булевая алгебра - способ работать с двумя значениями: 0 и 1,
которые интерпретируются как ложь и истина.
Есть четыре базовые операции:

1. НЕ
2. И
3. ИЛИ
4. ИСКЛЮЧАЮЩЕЕ ИЛИ (XOR)

Эти же операции применяются не к одному биту, а к **веторам битов**.
Это делается поэлеметно - каждый бит обрабатывается отдельно.

```
a & b:
0110
1100
----
0100   (1 только там, где оба бита = 1)

a | b:
0110
1100
----
1110   (1 там, где хотя бы один бит = 1)

a ^ b:
0110
1100
----
1010   (1 там, где биты различаются)

~b:
1100
----
0011   (инверсия каждого бита)
```
