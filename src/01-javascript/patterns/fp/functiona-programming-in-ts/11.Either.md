# Either

`Either` Позволяет сообщить, если функция выполнилась с ошибкой.

```ts
type Either<E, A> = Left<E> | Right<A>; // Если произошла ошибка, то возвращается Left, в другом случае Right

interface Left<E> {
  readonly _tag: "Left";
  readonly left: E;
}

interface Right<A> {
  readonly _tag: "Right";
  readonly right: A;
}

let result: Either<string, number>;
result = { right: 12 };
result = { right: -5 };
result = { left: "error" };
result = { left: "permission denied" };

// Helpers for creating left and right values

const left = <E, A = never>(e: E): Either<E, A> => ({
  _tag: "Left",
  left: e,
});

const right = <A, E = never>(a: A): Either<E, A> => ({
  _tag: "Right",
  right: a,
});
```

Пример использования до:

```ts
function divideTwoIfEvent(num: number): number {
  if (num === 0) {
    throw "cannot divide by zero";
  } else if (num % 2 !== 0) {
    throw "num is not event";
  } else {
    return 2 / num;
  }
}
```

Пример использования `Either`:

```ts
function divideTwoIfEvent(num: number): Either<string, number> {
  if (num == 0) {
    return left("cannot divide by zero");
  } else if (num % 2 !== 0) {
    return left("num is not event");
  } else {
    return right(2 / num);
  }
}
```

## Demo

```ts
type Increment = (x: number) => number;
const increment: Increment = (x) => x + 1;

const composed = compose(increment, divideTwoIfEvent); // Будет ошибка типизации. Надо использовать функтор. Временным решением может быть type guard
```
