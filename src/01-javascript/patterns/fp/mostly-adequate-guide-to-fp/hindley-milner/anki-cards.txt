Как читается сигнатура «String -> Number» в HM-нотации?	Функция принимает String и возвращает Number.	fp::hindley-milner::basics
Что означает запись «[a]» в HM-сигнатуре?	Массив элементов типа «a» (где «a» — типовая переменная, любой тип).	fp::hindley-milner::basics
Как в HM-нотации записывается каррированная функция двух аргументов?	a -> b -> c, что эквивалентно a -> (b -> c) — функция принимает «a» и возвращает функцию «b -> c».	fp::hindley-milner::currying
Что происходит с сигнатурой при частичном применении «match :: Regex -> String -> [String]» с одним аргументом?	«Откусывается» тип слева: результат имеет сигнатуру «String -> [String]».	fp::hindley-milner::currying
Чем отличается «a -> a» от «a -> b» в HM?	«a -> a» — вход и выход одного типа (одна буква = один тип). «a -> b» — вход и выход могут быть разных типов.	fp::hindley-milner::type-variables
Что такое параметричность (parametricity)?	Принцип, по которому функция с полиморфным типом не может инспектировать или менять значения абстрактного типа. Чем абстрактнее тип, тем меньше возможных реализаций.	fp::hindley-milner::parametricity
Почему «a -> a» допускает только одну реализацию (identity)?	Функция не знает, что такое «a», поэтому не может создать новое значение, преобразовать или проверить тип. Остаётся только вернуть аргумент как есть.	fp::hindley-milner::parametricity
Что может делать функция с сигнатурой «[a] -> [a]»?	Переставлять, удалять, повторять элементы — но не может создавать новые или менять существующие, потому что не знает тип «a».	fp::hindley-milner::parametricity
Что такое свободные теоремы Вадлера (theorems for free)?	Математические законы, которые автоматически следуют из сигнатуры типа без знания реализации. Например: compose(f, head) === compose(head, map(f)).	fp::hindley-milner::free-theorems
Какую практическую пользу дают свободные теоремы?	Бесплатные тест-кейсы (два эквивалентных выражения должны давать одинаковый результат) и возможности оптимизации (можно выбрать более производительный вариант).	fp::hindley-milner::free-theorems
Как читается ограничение «Ord a => [a] -> [a]»?	«При условии, что тип a поддерживает сравнение (тайпкласс Ord), функция принимает массив a и возвращает массив a».	fp::hindley-milner::type-constraints
Какой аналог «Ord a =>» в TypeScript?	<T extends Comparable> — ограничение generic-типа через extends.	fp::hindley-milner::type-constraints
Что является аналогом HM типовых переменных «a», «b» в TypeScript?	Generics: <T>, <A, B>. Например, «a -> a» соответствует «<T>(x: T) => T».	fp::hindley-milner::type-variables
Почему параметричность в JavaScript работает не полностью?	В JavaScript есть typeof, instanceof и приведение типов, которые позволяют «заглянуть» в абстрактный тип. В чистом HM это невозможно.	fp::hindley-milner::parametricity
Как записывается HM-сигнатура функции map?	map :: (a -> b) -> [a] -> [b] — принимает функцию преобразования и массив, возвращает массив преобразованных элементов.	fp::hindley-milner::basics
