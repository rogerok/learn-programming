# Упорядочивание асинхронных операций

Предположим, что перед нами стоит задача прочитать содержимое двух файлов и записать в третий.

```ts
import fs from "fs";

fs.readFile("./first", "utf-8", "?");
fs.readFile("./second", "utf-8", "?");
fs.writeFile("./new-file", content, "?");
```

Вся задача сводится к последовательному выполнению трех операций. Упорядочить подобный код можно лишь одним способом:
каждая последующая операция должна запускаться внутри коллбека предыдущей.

```ts
import fs from "fs";

fs.readFile("./first", "utf-8", (err1, data1) => {
  fs.readFile("./second", "utf-8", (err2, data2) => {
    fs.writeFile("./new-file", `${data1}${data2}`, (err3) => {
      console.log("File has been written");
    });
  });
});
```

В реальных программах подобных вызовов может быть десятки, и тогда мы сталкиваемся с callback hell.
