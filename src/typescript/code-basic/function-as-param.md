Как типизировать функции, которые передаются как параметры

```ts
function process(callback: Function) {
  const value = callback();
  // ...
}
```

Тип `Function` отключает проверку типов для вызываемой ф-ции. В итоге кол-во и тип входящих аргументов не проверяются, а результатом такого вызова будет `any`.
