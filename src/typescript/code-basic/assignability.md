# Присвоение значения

Присвоение одного значения другому и передача аргументом в функцию называют присваиваемостью(`Assignability`)

```ts
let x: number;
const y: number = 10;
x = y;

function len(str: string): number {
  return str.length;
}
len(false); // Error!
```

Если думать о типах, как о множествах значений, то присваиваемость - это проверка, что множество значений `x` входят в множество значений `y`

Переменная `x` присваивается переменной `y`, если множество значений `x` входит в множество значений `y`.
Или другими словами - если множество значений `x` является подмножеством значений `y`
